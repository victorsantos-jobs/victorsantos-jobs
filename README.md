## ⚡ About the Project

A **modern, secure and scalable REST API** built for **task and project management**, focused on **clean architecture, performance optimization, authentication, and enterprise-level backend practices**.

This project was designed with a strong emphasis on **modularity, maintainability, security, and production-ready structure**.

---

# ✨ Core Features

### 🔐 Authentication & Security
- JWT Authentication
- Role-Based Access Control (RBAC)
- Password Recovery & Email Verification
- Rate Limiting
- CORS Protection
- Helmet Security Headers
- Input Validation & Sanitization

---

### 📋 Task Management
- Create, update, delete tasks
- Priority & status tracking
- Task dependencies
- Comments & time logs
- File attachments
- Bulk updates

---

### 🚀 Project Management
- Team collaboration
- Milestones
- Risk management
- Attachments
- Progress monitoring
- Status tracking

---

### 📊 Analytics & Dashboard
- User statistics
- Task reports
- Priority insights
- Project overview
- Aggregations & optimized queries
